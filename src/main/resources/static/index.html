          概述 古镇渊源 古镇名人 古镇民俗

          古镇地处桐乡市北端，京杭大运河西侧，西临湖州市，北界江苏吴江县，为两省三市交界之处。作为首批中国历史文化名镇、中国十大魅力名镇、全国环境优美乡镇、国家5A级景区，古镇素有“中国最后的枕水人家”之誉。

          了解更多

          古镇拥有7000多年文明史和1300年建镇史。据古镇近郊的谭家湾古文化遗址考证表明，大约在7000年前，古镇的先民就在这一带繁衍生息了。古镇古名古墩、古戍。从地理、历史、典故都可找到相关记载，1950年，古、青两镇合并后称古镇至今。

          了解更多

          古镇的名人大家数不胜数，自古名人荟萃学子辈出，从一千多年前中国最早的诗文总集编选者梁昭明太子，到中国最早的镇志编撰者沈平等。近、现代更有中国第一任文化部长茅盾，海外华人文化界传奇大师孙木心等等。

          了解更多

          在古镇，有许许多多的传统节日，源远流长，久盛不衰，他们承载历史和文化。古镇当地百姓过的传统民俗节日，都有其特定的文化形象和节日主题，并且都与节令变化有关。
